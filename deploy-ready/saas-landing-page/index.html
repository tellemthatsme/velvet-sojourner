Frequently Asked Questions

What exactly am I paying for?

AgentForge is a deployment orchestration platform. You are paying for the infrastructure automation, unified dashboard, SSL management, and monitoring — not the underlying repositories, which remain open source.

Do I need the source code of these repos?

Yes. AgentForge generates deployment configurations, but you still need to clone the repositories yourself (or use your own GitHub tokens). We provide the infrastructure layer that makes deployment effortless.

Can I deploy to my own servers?

Yes. The Self-Hosted tier ($199 one-time) runs entirely on your infrastructure. The Managed Cloud tier ($49-499/mo) runs on our infrastructure with zero maintenance from you.

What if a repo doesn't have a Dockerfile?

AgentForge identifies 187 repos that are already Docker-ready. For repos without Dockerfiles, the platform generates a best-effort deployment config based on detected language and framework.

Is there a refund policy?

Yes. We offer a 30-day money-back guarantee. If AgentForge doesn't save you time, we'll refund you in full.
