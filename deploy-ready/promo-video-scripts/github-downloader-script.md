# GitHub Downloader — Promo Video Script (60s)

## Scene 1: Hook (0-10s)

**[Visual:**
- Close-up: someone tries to download a GitHub repo via the green "Code" button
- They click "Download ZIP" — file saves
- Cut to: they're on a CI server or a machine without git installed
- Text overlay: "No git. No problem."
- Logo reveal: GitHub Downloader — sleek desktop app icon + CLI badge
**]**

**[Audio:**
Energetic, driving electronic beat starts.
**Voiceover:** "Need to download a GitHub repo without git? We've got you covered."
**]**

---

## Scene 2: Problem (10-20s)

**[Visual:**
- Split screen montage:
  - Left: someone typing "git clone" in a terminal, getting "git is not recognized"
  - Right: someone manually clicking through a dozen repos on GitHub, downloading one ZIP at a time
  - Middle: a folder full of partially downloaded, misnamed ZIP files
- Text flashes: "There has to be a better way."
**]**

**[Audio:**
**Voiceover:** "Git's great — until you're on a machine without it. Or you need to grab a dozen repos for a project. Manually downloading from GitHub one at a time? That's not scale. That's suffering."
**]**

---

## Scene 3: Solution (20-40s)

**[Visual:**
- Screen recording of GitHub Downloader GUI launching:
  - Clean, modern interface with a large text input at top
  - Cursor pastes a GitHub URL (e.g., "https://github.com/user/repo") into the input
  - Presses "Download"
  - Progress bar animates smoothly
  - File saves to a "Downloads/Repos" folder
  - Cursor double-clicks — folder opens, repo contents visible
- Cut to: terminal window showing CLI usage
  - Types: `ghdownloader https://github.com/user/repo --output ./repos`
  - Progress bar in terminal
**]**

**[Audio:**
Music shifts to a productive, rhythmic tempo.
**Voiceover:** "Meet GitHub Downloader. Desktop app and CLI — two tools, one mission. Paste a URL, click download, done. The repo lands on your disk as a clean ZIP, ready to use. Need automation? The CLI handles batch downloads, CI/CD pipelines, and scripting."
**]**

---

## Scene 4: Features (40-55s)

**[Visual:**
- Quick feature tour:
  - Drag and drop a URL from browser into the GUI
  - Batch mode: pasting multiple URLs at once, all downloading in parallel
  - Right-click → "Open folder" on a completed download
  - Settings panel: choose output directory, naming convention
  - CLI flags scrolling: `--batch`, `--output`, `--quiet`, `--help`
  - Small badge: "Windows • macOS • Linux"
**]**

**[Audio:**
**Voiceover:** "Drag and drop. Batch download. GUI for browsing, CLI for automation. Cross-platform. One executable, zero dependencies. Download a repo in seconds, not minutes."
**]**

---

## Scene 5: Call to Action (55-60s)

**[Visual:**
- Full screen: GitHub Downloader logo
- Below: "ghdownloader.com" in bold
- Fade in download buttons: "Download for macOS" / "Download for Windows" / "Download for Linux"
- Small text beneath: "Free download. No account required."
**]**

**[Audio:**
Music builds to a confident finish.
**Voiceover:** "Download the GitHub Downloader at ghdownloader.com. Free. Fast. No git required."
