## Quick Start

1. Download `GitHubDownloader-GUI.exe`
2. Double-click to run (no installation needed)
3. Enter your GitHub token: Account → GitHub Authentication
4. Start downloading repos

## CLI Usage

```cmd
GitHubDownloader-CLI.exe download --url owner/repo -o C:\repos
GitHubDownloader-CLI.exe batch --file repos.txt -o C:\repos
```
