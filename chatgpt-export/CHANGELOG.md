# Changelog

## [1.0.0] - 2026-05-22
### Added
- Multi-platform support: ChatGPT, Claude, Gemini exports
- 5 input formats: TXT, MD, JSON, HTML, PDF
- Content-hash deduplication
- Topic classification (code, design, plan, guide, idea, general)
- Obsidian-compatible YAML frontmatter
- Merge broken conversation threads
- HTML and PDF output formats
- PyQt6 GUI with statistics dashboard
- CLI interface with --input, --output, --stats, --dedupe, --merge, --format flags
- OpenAI API integration for direct conversation export
- Semantic clustering using sentence-transformers
- SQLite knowledge base with full-text search
- Obsidian vault sync
- Comprehensive test suite (pytest)
