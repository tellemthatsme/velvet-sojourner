# lovestruck-crypto-nexus

## Description
TradingPro is a comprehensive cryptocurrency trading platform designed to provide traders with advanced tools for market analysis, portfolio management, automated trading, and AI-powered trading strat

## Tech Stack
JS/TS, Node.js, React, Next, Docker, GitHub Actions

## Quick Start
```bash
npm run dev  # vite
```
```bash
npm run build  # vite build
```
```bash
docker build -t <name> . && docker run <name>
```
```bash
docker-compose up
```
