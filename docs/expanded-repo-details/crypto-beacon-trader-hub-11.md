# crypto-beacon-trader-hub-11

## Description
A comprehensive cryptocurrency trading platform with AI-powered bots, real-time analytics, and portfolio management. Built for Australian traders with AUD as the primary currency.

## Tech Stack
JS/TS, Node.js, React, Next, Docker, GitHub Actions

## Quick Start
```bash
npm run dev  # vite
```
```bash
npm run build  # vite build
```
```bash
docker build -t <name> . && docker run <name>
```
```bash
docker-compose up
```
